Dharmasearch is a user interface for Dharma-api, the search engine on which this site depends. Dharma-api aims to have a growing number of sources which are at the heart of this site. Without the generousity of these sources and the hundreds of speakers whom have given talks, this site would not be possible. Please respect the licensing agreements that come with each talk and consider the spirit of dana (generosity).
